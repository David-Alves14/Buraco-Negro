# Buracos Negros

Projeto desenvolvido em HTML e CSS com o objetivo de apresentar informações sobre buracos negros de forma visual e interativa.

## Funcionalidades

* Menu de navegação com rolagem suave
* Seção explicando o que é um buraco negro
* Explicação sobre sua formação
* Curiosidades em formato de cards
* Vídeo incorporado com simulação 3D
* Design inspirado em sites de exploração espacial

## Tecnologias Utilizadas

* HTML5
* CSS3
